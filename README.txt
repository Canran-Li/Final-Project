1. install python moudle
   In terminal run
     sudo pip install matplotlib
     sudo pip intsall netCDF4
     sudo pip install os
     sudo pip install numpy
  to get needed moudle for GetData.py
2. Use pwd in terminal to get the path of GetData directory.
   Change the path of the fifth line of GetData.py to where you keep the GetData directory.
3. In terminal run
   cd GetData
   bash run.sh
   To get 24 data files, each one is named by its data name. The data files will be saved in Data directory.
